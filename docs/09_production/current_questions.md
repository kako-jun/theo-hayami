# 現代の問いコーナー

本編/自由行動で扱う「時代を超える根源的な業」と、サイトで継続更新する「現代的・時事的な悩み」は分ける。

本編の自由行動で選べる分母は、既に設計済みの根源的な業テーマを正本にする。現代の問いは混ぜず、サイト側の別コーナーとして扱う。

## 役割

現代の問いコーナーは、固定客を作るための継続更新枠。

- ユーザーから募った悩みを扱う。
- 各哲学者/住人の回答を後から作り足す。
- 全住人に必ず答えさせず、テーマごとに映える住人を選ぶ。
- 1テーマ1住人単位でショート動画化しやすくする。
- 本編の普遍性を壊さず、サイト側の更新性を担う。

## 待機列を公開する理由

扱う予定のテーマ待機列は、サイト上で公開する。

待機列は単なる内部 TODO ではない。

- ユーザーが今後の更新を見てワクワクできる。
- テーマ募集時の参考例になる。
- ユーザーが自分の悩みを投稿するときの例文として機能する。
- 「このサイトはこういう悩みを扱ってくれる」という期待値を作る。
- 投票/人気順/採用予定と組み合わせることで、再訪する理由になる。

本編の業テーマのような短い一語ではなく、ある程度長い悩み相談の質問文形式にする。

公開時は、原案の切実さは残しつつ、煽り・断定・特定集団攻撃に見えすぎる表現は少し中立化する。

## 待機列 原案

- ワールドカップで、くじ運や怪我や審判に泣かされて悔しい。最大のチャンスだった世代なのに、どう受け止めればいいのか。
- 天才にかなわないと感じるとき、どう生きればいいのか。
- AIに仕事を奪われるのが怖い。
- AIに人類が滅ぼされるのではないかと怖い。
- 告白するのが怖い。
- 大事な人と別れるのが怖い。
- 人生で何を残すべきなのか。
- 自分の先が見えた気がして怖い。
- 死ぬときの痛みが怖い。
- 出産が怖い。
- レベルの低いメディアの扇動を見るのがつらい。
- 戦争がなくならない理由は何なのか。
- お金持ちになりたい。
- レッドオーシャンに巻き込まれないためには、どうすればいいのか。
- 子育てに失敗したくない。
- すべての本を読めるわけでも、すべての国に行けるわけでもないことがつらい。
- 自分が生きている間に結果を見られないことに、なぜ人は興味を持てるのか。
- 多動症とどう付き合って生きればいいのか。
- 病気とどう付き合って生きればいいのか。
- 洗脳されたように、説得しようがないほど考え方が違う人とどう向き合えばいいのか。
- 軍がなければ攻められない、という考え方は本当なのか。
- 政治家や政党が、国益に反するように見える行動を取るのはなぜなのか。
- 烏合の衆が嫌いだ。集団の愚かさをどう受け止めればいいのか。
- 自分の生まれを呪ってしまう。
- ほとんどの人が先進国に生まれていないという現実を、どう考えればいいのか。
- 宇宙に必ず人間のような存在がいるわけではないという現実が怖い。
- 温度は高い方向にはいくらでもあるのに、低い方向は少ししかなく、自分たちはその付近にいるという不思議をどう考えればいいのか。
- 世界の国が一つになることは、本当に理想なのか。
- 診断されていないだけで、精神的に病んでいる人は多いのではないか。
- 滅ばない国は作れるのか。独裁国家が簡単には滅びそうにないのはなぜか。
- 政治や制度をハックするとは、どういうことなのか。
- 性欲やスキャンダルで有名人が消えていくのをどう見ればいいのか。
- 失敗したのに、失敗していないことにしたがる映画や作品をどう受け止めればいいのか。
- 差別された経験をどう抱えればいいのか。
- 商売や試験で勝ちたい。

## 置き場所案

本編:

```text
content/scripts/free/<業>__<住人>.md
content/scripts/main/*.md
```

現代の問い:

```text
content/scripts/current/<topic>__<住人>.md
```

またはサイト側だけで管理:

```text
content/site/current-questions/<topic>/<住人>.md
```

## 運用

- 本編自由行動の分母には混ぜない。
- サイトの「現代の問い」系コーナーとして扱う。
- 人気投票/投稿募集で上位のものから制作してよい。
- 質問文は長すぎず、ショート動画にも切り出しやすく整える。
- 採用済み/制作中/待機中が見えるようにする。
