# けふのティータイム

本編/自由行動で扱う「時代を超える根源的な業」と、サイトで継続更新する「現代的・時事的な悩み」は分ける。

本編の自由行動で選べる分母は、既に設計済みの根源的な業テーマを正本にする。現代的な問いは混ぜず、サイト側の `004 / けふのティータイム` として扱う。

## 役割

けふのティータイムは、固定客を作るための継続更新枠。

- ユーザーから募った悩みを扱う。
- 1つの問いについて、3人の住人による対談/情報交換形式を基本にする。
- せおは中心の聞き役として、問いを整理し、住人同士の視点の違いを引き出す。
- 全住人に必ず答えさせず、テーマごとに映える3人を選ぶ。
- 必要なら、対談内の1人分の発言をショート動画用に切り出せるようにする。
- 本編の普遍性を壊さず、サイト側の更新性を担う。

## 待機列を公開する理由

扱う予定のテーマ待機列は、サイト上で公開する。

待機列は単なる内部 TODO ではない。

- ユーザーが今後の更新を見てワクワクできる。
- テーマ募集時の参考例になる。
- ユーザーが自分の悩みを投稿するときの例文として機能する。
- 「このサイトはこういう悩みを扱ってくれる」という期待値を作る。
- 投票/人気順/採用予定と組み合わせることで、再訪する理由になる。

本編の業テーマのような短い一語ではなく、ある程度長い悩み相談の質問文形式にする。

公開時は、原案の切実さは残しつつ、煽り・断定・特定集団攻撃に見えすぎる表現は少し中立化する。

## 質問文化前の間引きルール

時事ネタはそのまま増やさず、質問文形式に整える前に既存の業テーマとの重複を判定する。

1. 候補ごとに、根っこの業へ写像する。例: 炎上で反論したい=論破/怒り、他人の成功がつらい=嫉み、AIに仕事を奪われるのが怖い=不安/焦り/意味。
2. 既存の業で十分扱えるものは、けふのティータイムの新規話題として増やさない。既存業への入口文に回すか、待機列から落とす。
3. 残すのは、AI、SNS、推し活、匿名コミュニティ、国際情勢など、現代的な表面や文脈がサイト導線として強いもの。
4. 判定表には `candidate / question_text / core_theme / decision(keep|merge|drop) / reason` を持つ。
5. 業と根が重なる場合でも、対談形式にすることで本編/自由行動と演出を変えられるかを確認する。変えられないなら作らない。

## 対談プロット

けふのティータイムは、採用時点で3人の選び方と分担まで先に決める。単独回答を3本並べるのではなく、時代を超えた哲人同士のやり取りをシミュレーションする。

- 住人A: 問いを切り分ける役。前提や言葉の混線をほどく。
- 住人B: 別の軸を持ち込む役。Aだけでは見えない価値・制度・身体・感情などを出す。
- 住人C: 現代への接続役。ユーザーが明日どう考えるかへ落とす。
- せお: 聞き役。問いを要約し、住人同士の差を確認し、結論を急がせない。

けんかや論破合戦にはしない。視点の違いは出すが、相手を潰すのでなく、問いを立体化するための対話にする。感情的になるのは、ちょっとだけにとどめる（語気を強める程度・相手を否定しない）。

- せお／ヴィンチアは対談の必須要員ではない。特別扱いせず、住人と同格の1人として参加してよい回もある。せお・ヴィンチアを入れない回は、聞き役不在のぶん3住人がそのまま視点をぶつけ合う（例: 第一弾「大会の悔しさ」）。
- せお1人を説得して終わる自由行動のセルとは尺の作りが違う。3人がそれぞれの軸で話すぶん、多少長めになってよい。
- 平坦に進めない。中盤で急展開・どんでん返し・「終わったと思ったところからの延長」のどれかを最低1つ入れ、退屈させない。
- 結論はいちおう出す。なるべくポジティブに着地させる（3人の結論を1つに丸めなくてよいが、後味は前向きにする）。

各ネタの制作前に、最低限この形でプロットを作る。

```text
topic:
question_text:
core_theme:
decision: keep|merge|drop
reason:
participants:
  - resident:
    role:
    facet:
    contrast_with:
  - resident:
    role:
    facet:
    contrast_with:
  - resident:
    role:
    facet:
    contrast_with:
seo_role:
conversation_arc:
short_video_cut:
```

- 政治家や政党が国益に反するように見える行動を取るのは、なぜでしょうか。

- 世界の国が一つになることは、本当に理想なのでしょうか。

## 置き場所案

本編:

```text
content/scripts/free/<業>__<住人>.md
content/scripts/main/*.md
```

けふのティータイム:

```text
content/scripts/current/<topic>__<住人>.md
```

またはサイト側だけで管理:

```text
content/site/current-questions/<topic>/<住人>.md
```

## 運用

- 本編自由行動の分母には混ぜない。
- サイトの `004 / けふのティータイム` として扱う。
- 人気投票/投稿募集で上位のものから制作してよい。
- 質問文は長すぎず、ショート動画にも切り出しやすく整える。
- 採用済み/制作中/待機中が見えるようにする。
