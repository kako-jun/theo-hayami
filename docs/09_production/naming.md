# 命名の由来

- 表示名 **せおはやみ**（かな）／ repo・サブドメイン **`theo-hayami`**。
- **せをはやみ** ＝ 崇徳院・百人一首「瀬をはやみ岩にせかるる滝川のわれても末に逢はむとぞ思ふ」。歌意（急流に岩で隔てられても末に逢う）が企画と一致＝**時代という急流に隔てられた読者と、何百年前の知性を、もう一度逢わせる**。物語の「消失と再会」「答えを拒否する」テーマとも響く。
- **せおはやみ** ＝ セオリー＋早見（表）。名前自体が「哲学の早見表」＝シリーズの使命を白状している。
- **theo-hayami** ＝ theory(theo) ＋ 早見(hayami)。"Theo Hayami" で実在しそうな人名にも見える。
- シリーズ：「わかりにくいものをわかりやすく」（**osaka-kenpo の兄弟**）。
