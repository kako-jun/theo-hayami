# バックログ業 差異化プロット（住人 × 業）— 記憶that新鮮なうちに固定した collision 予防の正本

> **目的**: 第一バッチ8業の外＝バックログ約29業について、**各住人that・どの業で・どの思想概念と比喩で答えるか**を先に確定する。脚本はまだ書かない。**プロットthat非かぶりを固定しているので、脚本化はセッションを分けても安全**（kako-jun 指示「かぶり予防はプロットでかなり防げる／記憶thtあるうちに作る」2026-06-27）。
> **作り方**: 住人ごとに1体のサブエージェントthat、自分の思想家MD＋[`concept_usage_ledger.md`](concept_usage_ledger.md)の既使用概念を入力に、未使用概念を非かぶりで割当（行=住人方向の collision を最小化）。各思想家の「誤解されやすい点」を踏まないことも確認済み。
> **使い方（脚本化時）**: 書く業を決めたら、下の**8つの表からその業の行を拾い**、`/theo-hayami-write theme <業>` の各 subagent に「概念・切り口・決め台詞核・主要metaphor」を渡す。本編おはこ除外（幻想=デカリス／あたりまえ=ヒュー／分けること=アリスト）に注意。書いたら ledger に追記。
> **状態**: プロット=全29業確定。脚本化=後半6業（変化/幻想/正しさ/迷い/あたりまえ/分けること＝計45セル）を 2026-06-30 に本番化完了（free/＋hub配線＋台帳追記）。老いは第一バッチ外だが既に脚本化済み・別管理。
> **2026-06-30 追記**: 後半6業（変化/幻想/正しさ/迷い/あたりまえ/分けること）の脚本化時に、(A) プロット欠落4セルを台帳照合のうえ補填＝カンティア×分けること（規定的判断力）・ヘグル×迷い（判断と推論）・ヘグル×分けること（区別と同一性）・スピノ×幻想（真理は自らの基準 II p43）。(B) 既存プロットの台帳衝突6件を訂正＝カンティア×迷い比喩（鋳型→楽譜のない調べ・依存と衝突）／スピノ×正しさ比喩（物差し→一枚の地盤・hue正しさ定規と列内衝突）／オウ×変化（為善去悪の決め台詞を幸福と差別化）・オウ×幻想（心外無物→本心本体・退屈と衝突）・オウ×あたりまえ（性即理→格竹・正しさと棲み分け）・オウ×分けること（四有四無→是非の心・同調と衝突＝台帳の警告どおり）。オウは在庫枯渇（在庫メモ既述）のため各セルを別ファセット＋別比喩で弁別。

口語タイトル⇔業の対応は [`combination_grid_drafts.md`](combination_grid_drafts.md) の「全テーマの口語タイトル」を参照。
