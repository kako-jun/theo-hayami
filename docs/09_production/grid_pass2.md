## かぶりチェック結果（全38テーマ×8住人を列照合・2026-06-26）

### 結論

- **行（住人ごと）＝解決済み**。各住人が「どのテーマでも結局同じ技」になる問題は消えた。知識MDの広い在庫で、テーマごとに別著作の別概念を当てられた（各エージェント report も行内再登場ほぼゼロ）。**これが本丸の達成。**
- **列（テーマごと）＝"哲学の自然な収束"が残る**。8思想家いると、構造的テーマで2〜3の「思想家ファミリー」に割れる。**同一セル（入替可能）ではない**（機構・比喩・せおの着地が違う）が、**隣接**はする。8人でこれをゼロにはできない。
- 対処方針: (a) 真に入替可能な**ハード衝突は書き分けを明記して潰す**。(b) **隣接クラスタは差分軸を下記に記録**＝脚本化時にこの軸で機構・比喩を振り分ける。(c) **混雑テーマ（時間・焦り 等）は全8セルを本番化しない**＝映える3〜4人だけ作る（「映えるセルだけ」の原則がそのまま効く）。

### ハード衝突（入替可能寸前→書き分けを固定）

- **傲慢**: カンティア(超越論的仮象「全体を掴んだと誇る越権」) ≈ ヘグル(絶対知「全部わかったを所有と取り違え」)。両方「完全知の僭称」。→ **書き分け固定**: カンティア＝*理性の越権・境界*（掴めない領域に手を出す錯覚）／ヘグル＝*知は所有でなく運動*（自覚を持ち物と取り違えた顔）。机上の論点を別にする。
- **依存**: デカリス(実体「完全な自立を欲しがるのは思い上がり」) ≈ スピノ(自己原因「他に依らず在るは神だけ」)。両方「自立できるのは神のみ」。→ **着地が別**なので両立可: デカリス＝*craving を叱る*（自立を欲しがるな＝範疇錯誤）／スピノ＝*依存を肯定*（依存は有限者の事実、問いは「何に依るか」だけ）。せおのお持ち帰りが逆向き。

- 怠け｜機能論証(ergon)｜人間固有の機能=理性的活動を遊ばせておくこと
- 見栄｜説得の三様式(ethos)｜人柄への信頼を、中身でなく見せかけで作る
- 孤独｜政治的動物(zōon politikon)｜論じ合う相手を欠く孤独は欠落、神か獣でなければ一人で完結せぬ

### カンティア（カント）
