# 配役の選考基準（設計記録）

レギュラー8の選び方。人格化（改名）する前の、実在ベースの素性で選んだ：

1. **自前の体系を一個立てた**（名前が -ism／学派になったか、がリトマス。例：マキャベリズム）
2. **著作が多い**（正確に引ける材料が増える＝実利）
3. **世俗・没後100年超・攻撃的な estate なし**
4. **既に散々キャラ化されていない**（比較されない）

※ 最終的に「思想の人格化＋改名（カンティア等）」にしたことで、estate・肖像権・名誉毀損は**二重に回避**される。ただし配役の素性は上記基準で選んでいる。

## 検討して外した人物（記録）

ソクラテス・プラトン（本人が嫌い）／ブッダ（宗教）／西田幾多郎（1945没・遺族/記念館）／ニーチェ（既に過剰にキャラ化＝比較される）／アインシュタイン（遺族が肖像権を厳格管理＝司会候補から外しダ・ヴィンチへ）／マルクス・アウレリウス（ストアを自分で作っていない＋著作が『自省録』一冊で薄い）／孔子（"述べて作らず"で体系の自前性が弱い、王陽明に席を譲る）／孫子・杜甫（孫子は兵法という体系を持つが著作が薄い、杜甫は詩人で枠組みでなく感受性）。

「各文明1人ずつ」案は、この体系構築者基準に上書きされた（結果、顔ぶれは西洋古典＋王陽明）。
