# 概念・比喩 使用台帳（住人 × 業）— かぶり予防の正本

> **目的**: 各住人that「どの業で・どの思想概念と・どの比喩を使ったか」を記録し、**新しい業を書くとき、その住人の既出概念・比喩の再使用を防ぐ**。セッションをまたいでも（書き手の文脈that消えても）この表を読めば collision 予防thtできる。
> **運用**: `/theo-hayami-write` で新しい業を書く前に、各住人の行を読み、未使用の概念を割り当てる。書いたら**この表に追記する**（concept＋主要metaphor）。
> **凡例**: 【本編】＝その業×住人は main/おはこthat担い free を書かない。**スピノの海/波は本編「死」専用＝全 free 禁止**。

## 新しい業を書くときの手順（collision 予防）
1. この表で、起稿する業の**各住人の行を確認**し、まだ使っていない概念・比喩を割り当てる。
2. 思想家 MD（`thinkers/{slug}.md`）の概念インベントリから、未使用で業に最も合うものを選ぶ。
3. 書いたら**この表に1行追記**（概念＋主要metaphor）。これを怠るとセッションをまたいだ時にかぶる。
4. 概念の在庫that尽きてきた住人（特に複数の業で同系統に寄る kantia=目的論系・dekaris=機械/二元系・hue=動機づけ系・ou=心本体系）は、MD の別系統から意図的に引く。
