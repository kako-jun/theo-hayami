# theo-hayami（せおはやみ）docs

このリポジトリの設計の正本。2つの層からなる:

1. **物語層（story bible v2 / ChatGPT で深掘り）** — 本 docs の `00_master`〜`08_unused`。世界・キャラ・四幕・伏線・真相。
2. **制作層（前段の対話で決めた仕様）** — 脚本MD形式／振り付け／画面（name-name 埋め込み）／立ち絵“こびない”方針／口調／命名由来／配役の選考基準。`09_production/` に格納済み。

## 一言で

盲目の主人公せおはやみが「叡智の星海」で、思想が人格化した住人たちに学び、最後に**与えられた“答え”を拒否して**自分で考えることを選ぶ物語。ヴィンチア（＝人類最初の「知りたい」衝動の人格化）が司会。エンディングの一文は **「この物語で語られたことを、まず疑え」**。

## 前段の対話設計との「すり合わせ」（解決済み 2026-06-18）

1. **名前 → カタカナ人格化名（B）で確定。** カンティア等。estate/肖像権を回避し、元ネタも透ける。実在名はこれに上書き。
2. **叡智 → 漢字2文字の概念（A）。** 認識・懐疑・弁証…。思想が“叡智”として発動（ペルソナ/スタンド的、**バトルにはしない**）。→ `02_characters/character_bible.md`
3. **形式 → ハイブリッド確定。** 四幕の物語＋ペルソナ/ダンガンロンパ式の**自由行動パート**。前段の〔お題×住人〕作り置きアンソロジー（笑える engine）は、自由行動パートの中身として生きる。→ `03_story/structure_hybrid.md`
4. **二層を貫く仕掛け。** 自由行動で親近感を育てた住人が、物語の進行で消え、「もう聞けない」喪失を生む＝**普通のひな壇でない意外性**。
