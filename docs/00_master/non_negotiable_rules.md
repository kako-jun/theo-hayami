
# 絶対に変更してはいけない事項

- 主人公は盲目
- ヴィンチアは黒幕ではない
- 黒幕は存在しない
- 最後に答えを得てはいけない
- 哲学者本人は登場しない
- 思想の人格化である
- エンディング文言『この物語で語られたことを、まず疑え』
- 答えの本を読ませない
- 最終的な正解を提示しない
- 主人公を救世主にしない
- 哲学者や思想を最終正解にしない
- **方言は使わない（全員 標準語）。** 影の図書館は無時間・無場所の抽象空間＝思想の人格化であり、特定の土地・人格に紐づく方言は世界観を壊し、estate回避（本人ではない）も損なう。声の作り分けは一人称／二人称／語尾／温度（register）で行い、accent（土地）に頼らない。大阪弁は兄弟作 osaka-kenpo の領分で、混ぜると両作の輪郭がぼやける。（声の割り当ては `02_characters/character_bible.md` の口調表が正本）
