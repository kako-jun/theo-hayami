# 構造：ハイブリッド（物語 × 自由行動）

二層が噛み合う。物語が背骨、自由行動が肉。

## メインストーリー（四幕・線形）

影の図書館での進行。盲目のせおはやみが住人に学び、住人が一人ずつ消え、ヴィンチア黒幕説を経て、最後に「答えの本」を**拒否する**。（→ `03_story/four_act_structure.md`）

## 自由行動パート（ペルソナ／ダンガンロンパ式）

ストーリーの合間に、せおはやみが **8人の誰にでも**、「**時を超える人間の業**」テーマを自由に聞ける。

- テーマ例：論破／妬み／吊し上げ（私刑）／儲け／承認／同調圧力／アバター／AI…（**固有名詞ゼロ・暗示のみ**、人格でなく公開された型を狙う）
- 各〔テーマ × 住人〕＝**作り置きの脚本MD**。前段で設計した engine「古い体系を現代に“わざと専門外で”大真面目に当てはめる（ズレ＝笑い＋発見）」が、ここに生きる。
- 住人は人格化名（カンティア等）で答え、思想を能力（認識・懐疑等）として振るう。説教でなく腕を上げてやる師匠。

## 二層を貫く仕掛け：ひな壇が、死ぬ

**最大の意外性。** 自由行動で親近感を育てた相手が、物語の進行で**いきなり消える**。

- 「このテーマをスピノに聞きたかったのに、もういないのか」——**聞けたはずの選択肢が、永久に閉じる**。
- 自由行動の顔ぶれ（＝聞ける相手）が、物語が進むほど **減っていく**。
- だから普通のひな壇（全員いつもいる）でなく、**喪失と不可逆**を抱えた盤面になる。「早く聞いておけば」という後悔が燃料。
- スピノが最初に消えるのは伏線と一致（スピノだけ死を恐れない＝汎神論 / キービジュアル「スピノ消失」）。百人一首「われても末に逢はむ」＝隔たりと再会のテーマとも響く。
