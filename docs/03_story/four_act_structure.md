
# 四幕構成

第一幕: 図書館との出会い
第二幕: 最初の消失
第三幕: ヴィンチア黒幕説
第四幕: 答えの本と卒業

## 本編フック案: 幻の11人目

序盤〜中盤のどこかで、せおが「ひとり少ないぞ。10人しかいない！」という台詞に触れる。元ネタは「11人いる！」の逆向きパロディ。

現在の登場人物は、せお + 8住人 + ヴィンチア = 10人。そこに「影の図書館には、幻の11人目の哲学者がいるらしい」という噂を置く。ただし実際にレギュラーを1人増やすのではなく、物語を進めるマクガフィンとして扱う。

機能:

- 10人構成そのものを、作中で不穏な違和感として読者に意識させる。
- ヴィンチア黒幕説や住人消失と絡め、「数が合わない」感覚を増幅する。
- 最終的には、11人目を探す話ではなく、欠けているものを「答え」で埋めようとする誘惑の比喩に接続できる。

全体テーマ:
答えを得る話ではなく、自分で考える話。

終盤の核:

1. 最深部で「完全なる答えの本」に到達する。
2. 読者は幸福や人生の答えを想像するが、実際は宇宙そのものの設計図である。
3. その答えは完璧だが、「なぜ」は説明しない。
4. せおはやみは本を閉じる。
5. 卒業とは、答えを信仰せず、問いを持ち帰ること。
