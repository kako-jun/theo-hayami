
# 理系設定

プラトンのイデア論、ホログラフィック宇宙論、大統一理論／万物の理論を物語内で統合する。

## 統合の芯

- イデア界は宗教的な天国ではなく、宇宙の情報層・ソースコード。
- 現実世界はその投影。
- プラトンの洞窟の比喩とホログラフィック宇宙論は、同じ構造を別言語で説明している。
- **大統一理論（力の統一を目指す実在の物理の探究）を、哲学の「一なるイデア／善へ登る」道と同じ山の別ルートとして絡める**（「哲学と物理は同じ山を別ルートで登る」の具体化）。住人＝思想のイデア（哲学ルート）と、理系レイヤー（大統一理論・ホログラフィック宇宙）が、頂上＝**完全なる答えの本＝完成した統一の数式**で合流する。
- 万物の理論は、物理だけでなく思想・意識・時間・死・自由意志まで説明しようとする（大統一理論を力の統一を超えて万物へ延長した究極形）。

## 限界

万物の理論は「あらゆる仕組み」を説明できるが、「なぜ」を説明できない。宇宙がどう動くかを説明しても、なぜ宇宙があるのか、なぜ美しいと思うのか、なぜ生きたいのかは残る。

この限界が、答えの本を閉じる理由になる。
