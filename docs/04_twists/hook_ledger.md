# 本編フック管理台帳

本編で読者に持たせる疑問を、提示・誤誘導・回収まで管理する。ここに載せたフックは、脚本内で不用意に答えを早出ししない。

## フックの分類

読者の注意を引く要素は、**謎・伏線・ノイズ**に分けて管理する。全部を回収すると世界が狭くなる。全部を回収しないと雑になる。回収契約の有無と距離を明示し、短い回収・中距離回収・ロングパス・未回収の外連味を混ぜる。

| 種類 | 役割 | 回収契約 | 使い方 |
|---|---|---|---|
| 謎 | 読者に「なぜ/誰/何が起きた」を考えさせる | 必ずある | M1〜M8の主フック。提示、誤誘導、部分開示、最終回収を管理する |
| 伏線 | 後で意味が変わる具体物・台詞・行動 | 必ずある | 短距離からロングパスまで混ぜる。回収時に「そういう意味だったのか」を作る |
| ノイズ | 世界が広く見える錯覚、未踏領域、外連味 | 原則なし | 最後まで明かさなくてよい。ただし主フックと矛盾させない。読者が本筋の答えだと誤認しすぎるものは置かない |

### 回収距離

| 距離 | 目安 | 効果 | 偏りすぎた時の問題 |
|---|---|---|---|
| 即時回収 | 同じmd内 | 読みやすさ、気持ちよい小解決 | 予定調和になる |
| 短距離 | 次md〜同幕内 | 連載の推進力、安心して読める手応え | 先が読めすぎる |
| 中距離 | 次幕〜2幕後 | 推理の持続、誤誘導の余地 | 忘れられると効かない |
| ロングパス | 終盤回収 | 大きなカタルシス、再読性 | 多すぎると現在のmdが未払いだらけになる |
| ノイズ | 回収しない | 世界観の奥行き、外連味 | 多すぎると雑な未回収に見える |

## 面白さの管理項目

謎そのものとは別に、各mdを書く前に以下を決める。ここを空欄にすると、本文は説明か雰囲気だけになりやすい。

| 管理項目 | 目的 | 書く前に決めること |
|---|---|---|
| 引きの強度 | mdを分ける意味を作る | そのmd単体で次を読みたくなる未完了感があるか。第一幕末は最強にする |
| 緊張源 | 読者の注意を引き締める | 何が不穏なのか。空席、返却口、沈黙、数のずれなど、具体物で持たせる |
| 緩和源 | 緊張を効かせる余白を作る | 安心、親しさ、自由行動解放、軽い会話など、読者が一度息をつける場所を置く |
| 読者に残す問い | 読後の頭の中を設計する | 「誰が」「なぜ」「何が起きた」のうち、どれを残すか |
| 誤誘導の方向 | 推理の足場を与える | 死、忘却、ヴィンチア黒幕、11人目実在など、どこへ疑わせるか |
| カタルシス予約 | 後の反転を気持ちよくする | 誤誘導が崩れたとき、何が「そういう意味だったのか」に変わるか |
| 回収契約 | 後で払う約束を明確にする | 示したものを、どの幕のどのmdで意味変換するか |
| 禁止する答え | ネタバレを防ぐ | そのmdでは絶対に言わない真相・用語・因果を決める |
| 感情の出口 | 説明で終わらせない | 安心、不穏、喪失、疑い、選択など、読者の最後の感情を決める |
| 物証 | 謎を会話だけにしない | 空席、名札、返却口、しおり、灯りなど、読者が覚えられる物を置く |
| 次mdへの橋 | 連載の推進力を作る | 次に確認したい対象を一つだけ立てる |
| 読ませる力 | 続きを開く欲を明示する | 推理欲、保護欲、喪失回避、関係性の緊張、絵を見たい欲、官能の予感など、何で次を押すか |
| 想定読了時間 | mdの重さを制御する | 何分読ませるつもりか。山場以外は同じ長さに揃え、幕末だけ体感速度を上げる |
| 読者知識 | 作者の全知を防ぐ | その時点で読者が知っていること、知らないこと、誤って信じてよいことを分ける |
| 絵の役割 | 画像を報酬と伏線にする | 立ち絵、イベント絵、崩壊背景、差分演出のどれを使い、読ませる力にどう寄与するか |

### 絵の使い分け

| 種類 | 使い所 | 使いすぎ防止 |

## 緊張と緩和 / 誤誘導とカタルシス

各幕は「緩和で油断させ、緊張で引く」「誤誘導で読ませ、カタルシスで反転する」を持つ。緊張だけを積むと疲れる。緩和だけだと散る。誤誘導だけだと騙しになる。カタルシスだけだと都合がよく見える。

| 幕 | 緩和 | 緊張 | 誤誘導 | カタルシス |
|---|---|---|---|---|

## 起承転結 / 局所的な序破急 / クライマックス管理

全体構造は **起承転結** を採用する。theo-hayami は、転のあとにすぐ結ぶより、エンディング前の弛緩・後奏・小さな着地を必要とするため。**序破急** は全体構造として併用しない。幕末や対峙など、局所的に「転したらすぐ結へ畳む」場面の加速記号としてだけ使う。

各mdは情報量ではなく、音楽の小節のように強弱を持たせる。すべてのmdを同じ強さで書かない。`強度` は 1〜5 で管理し、5 は幕のクライマックスだけに使う。

| 区分 | 役割 | 使い方 |
|---|---|---|
| 起 | 場・問い・期待を立てる | 新しい状況を読者が理解できるようにする。まだ答えを急がない |
| 承 | 期待を広げ、親しませる | 住人・場所・問いを好きにさせる。後の喪失の土台にする |
| 転 | それまでの読みをずらす | 読者の仮説を揺らす。物証を置き、説明は遅らせる |
| 結 | 感情を固定して次へ渡す | 解決ではなく、幕末なら強い未完了感を残す |
| 局所的な序 | 小さく入り直す | 山場前に足場を作る。全体の起とは別 |
| 局所的な破 | 小さな均衡を壊す | 違和感、矛盾、失踪、疑いを入れる |
| 局所的な急 | すぐ結へ押し出す | 幕末・対峙・真相で読む速度を上げる |

| 題のない本 | `act1-04` | スピノの本、または11人目の本 | 消失者の記録 | 答えの本と対になる、まだ題を持たない問い |
| しおり | `act1-04` | 読みかけの印 | 消えた瞬間の保存 | せおが次に問うべき場所を示す印 |
| 波の音 | `act1-04` | スピノの消失予告 | 死や海への回帰 | 「すべては一つ」へ近い思想だったことの後出し回収 |

## 第一幕で守ること

- 11人目は示唆するが、マクガフィンだとは明かさない。
- 消失は示唆するが、理由を説明しない。
- ヴィンチアは怪しさと親しさを両方残す。
- 「答えの本」はまだ強く出さない。第一幕の読後フックは「空席」と「消える噂」で十分。
- 読者に持たせる疑問は、答えでなく観察対象として渡す。

## 幻の11人目

| 項目 | 内容 |
|---|---|
| 読者に持たせる疑問 | 影の図書館には、数に入っていない誰かがいるのか。せおの数え間違いなのか。 |
| 正体 | 実在する追加レギュラーではない。欠けているものを「答え」で埋めたくなる誘惑のマクガフィン。 |
| 提示 | 第一幕末。8住人に会った後、せおが「数は合うのに足りない」と感じる。 |
| 誤誘導 | 第二幕以降、住人の消失と結びつき、「消えた誰か」「ヴィンチアが隠した誰か」に見える。 |
| 回収 | 終盤、11人目を見つける話ではなく、欠けを答えで埋めようとする誘惑そのものだったとわかる。 |
| 禁止 | 本当に11人目の哲学者を追加しない。早い段階で「ただのマクガフィン」と明かさない。 |

## 住人の消失

| 項目 | 内容 |
|---|---|
| 読者に持たせる疑問 | なぜ住人が消えるのか。死なのか、忘却なのか、ヴィンチアが消しているのか。 |
| 正体 | 住人は思想の梯子として役目を終える。せおが登ったぶん、外に立っていた梯子は不要になる。 |
| 提示 | 第一幕では「ここでは、ときどき誰かがいなくなる」という噂・気配だけ。理由は言わない。 |
| 誤誘導 | 第二幕から、スピノの消失を「死」「忘却」「図書館の危険」「ヴィンチア黒幕説」に見せる。 |
| 回収 | 第三幕後半。黒幕説が崩れ、消失が喪失ではなく卒業に反転する。 |
| 禁止 | 第一幕で「考えるのをやめると消える」と言わない。消失理由をルール説明にしない。 |

## ヴィンチア黒幕説

| 項目 | 内容 |
|---|---|
| 読者に持たせる疑問 | ヴィンチアはなぜ全部知っているのか。なぜ消えないのか。住人消失に関わっているのか。 |
| 正体 | 黒幕ではない。答えではなく問いを守る好奇心の人格化。 |
| 提示 | 第一幕から、ヴィンチアは噂を知っているが説明しきらない。 |
| 誤誘導 | 消失が進むほど、説明を避ける案内人として怪しく見せる。 |
| 回収 | 第三幕。疑いをぶつけ、黒幕ではなく問いの側にいるとわかる。 |
| 禁止 | ヴィンチアを早期に善人として保証しない。逆に悪意ある黒幕にも確定させない。 |

## 答えの本

| 項目 | 内容 |
|---|---|
| 読者に持たせる疑問 | 最後に得られる答えは、人生の答えなのか、幸福の答えなのか。 |
| 正体 | 宇宙の設計図。完全な説明ではあるが、「なぜ」を持たない。 |
| 提示 | 序盤では「答え」への欲望を直接出さず、11人目・消失・黒幕説の欠けを通じて育てる。 |
| 誤誘導 | 欠けた数、消えた住人、隠された真相を、ひとつの答えで埋められるように見せる。 |
| 回収 | 第四幕。答えの本を閉じることで、答えではなく問いを選ぶ。 |
| 禁止 | 早い段階で「答えの本が罠」と明かさない。 |
