# 本編フック管理台帳

本編で読者に持たせる疑問を、提示・誤誘導・回収まで管理する。ここに載せたフックは、脚本内で不用意に答えを早出ししない。

## 分母と進捗

読者に興味を持たせるための「謎」は、現時点で **8本** に絞る。増やす場合はこの表に足し、既存の謎を圧迫しないか確認してから本文へ入れる。

| 状態 | 本数 | 定義 |
|---|---:|---|
| 未提示 | 1 | 本編にまだ影も出していない |
| 提示済み | 4 | 読者が疑問として持てる状態 |
| 部分開示 | 3 | 一部の答えは見えたが、核心は未回収 |
| 回収済み | 0 | 作品内で答えを渡した |
| 保留/削除候補 | 0 | 使わない可能性が高い |

現在の第一幕末時点で、読者に推理させる謎は **4本まで**。M5/M6/M8は舞台理解のために部分開示するが、主フックにはしない。第一幕は世界観説明が多いため、主フックをこれ以上増やすと散る。第二幕以降で残りを提示し、第三幕で最大の誤誘導、第四幕で回収する。

## 第一幕で守ること

- 11人目は示唆するが、マクガフィンだとは明かさない。
- 消失は示唆するが、理由を説明しない。
- ヴィンチアは怪しさと親しさを両方残す。
- 「答えの本」はまだ強く出さない。第一幕の読後フックは「空席」と「消える噂」で十分。
- 読者に持たせる疑問は、答えでなく観察対象として渡す。
