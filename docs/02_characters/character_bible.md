# キャラクターバイブル

哲学者本人は登場しない。**思想の人格化**。住人は一人が「人格化名（呼び名）」と「能力名（思想の技）」を二枚持つ。

## 住人8人

| 人格化名（B） | 能力名（A） | 思想 | 由来（実在） |
|---|---|---|---|
| アリスト | 観察 | 分類と観察 | アリストテレス |
| カンティア | 認識 | 認識論 | カント |
| ヘグル | 弁証 | 弁証法 | ヘーゲル |
| デカリス | 懐疑 | 方法的懐疑 | デカルト |
| スピノ | 汎神 | 汎神論 | スピノザ |
| ヒュー | 経験 | 経験論 | ヒューム |
| マキヤ | 現実 | 現実主義 | マキャベリ |
| オウ | 知行 | 知行合一 | 王陽明 |

- **名前＝カタカナ人格化名（B案）**。耳には“人”だが本人ではない＝estate（遺族・肖像権）を回避。元ネタが透けて「あ、あの人だ」の発見も残る。
- **能力名＝漢字2文字の概念（A案）**。思想が“技”として発動する（ペルソナ／スタンド的）。**バトルにはしない**（ボツ案「哲学バトル」を尊重）＝思想の方法を物語内で振るう（例：デカリスが「懐疑」で幻を解く、スピノが「汎神」で死を超える）。
- 能力名は思想の2文字短縮（暫定。観察/認識/弁証/懐疑/汎神/経験/現実/知行）。要なら磨く。

## 主役・司会

- **せおはやみ**：盲目の主人公・読者の分身。中性・無色・聞き役。**能力を持たない**（普通の人間＝読者の目だから）。視覚描写を避ける＝盲目の伏線、「夢で初めて色を見る」で回収。
- **ヴィンチア**：司会。人類最初の「知りたい」衝動の人格化。**黒幕ではない**（絶対ルール）。
