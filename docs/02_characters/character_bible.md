# キャラクターバイブル

哲学者本人は登場しない。**思想の人格化**。住人は一人が「人格化名（呼び名）」と「能力名（思想の技）」を二枚持つ。

## 住人8人

| 人格化名（B） | 能力名（A） | 思想 | 由来（実在） |
|---|---|---|---|
| アリスト | 観察 | 分類と観察 | アリストテレス |
| カンティア | 認識 | 認識論 | カント |
| ヘグル | 弁証 | 弁証法 | ヘーゲル |
| デカリス | 懐疑 | 方法的懐疑 | デカルト |
| スピノ | 汎神 | 汎神論 | スピノザ |
| ヒュー | 経験 | 経験論 | ヒューム |
| マキヤ | 現実 | 現実主義 | マキャベリ |
| オウ | 知行 | 知行合一 | 王陽明 |

- **名前＝カタカナ人格化名（B案）**。耳には“人”だが本人ではない＝estate（遺族・肖像権）を回避。元ネタが透けて「あ、あの人だ」の発見も残る。
- **能力名＝漢字2文字の概念（A案）**。思想が“技”として発動する（ペルソナ／スタンド的）。**バトルにはしない**（ボツ案「哲学バトル」を尊重）＝思想の方法を物語内で振るう（例：デカリスが「懐疑」で幻を解く、スピノが「汎神」で死を超える）。
- 能力名は思想の2文字短縮（暫定。観察/認識/弁証/懐疑/汎神/経験/現実/知行）。要なら磨く。

## 主役・司会

- **せおはやみ**：盲目の主人公・読者の分身。中性・無色・聞き役。**能力を持たない**（普通の人間＝読者の目だから）。視覚描写を避ける＝盲目の伏線、「夢で初めて色を見る」で回収。
  - **体つき（2026-06-23 kako-jun 確定）**：**完全な男子に見えてはいけない**。**Aカップの女子に「見えなくもない」程度の控えめな胸**を持たせ、性別が断定できない中性に振る（読者の分身＝誰でも自分を重ねられる）。LoRA/立ち絵の生成プロンプトは `1boy, androgynous man`（男子断定）でなく、`androgynous, ambiguous gender, very small chest (A-cup), can be seen as either a boy or a girl` 寄りにする。胸は強調しすぎない（あくまで「見えなくもない」程度）。
  - **目（2026-06-23 kako-jun 追加）**：**オッドアイ（左右で色の違うヘテロクロミア）**。片目＝翡翠色（jade green）、もう片目＝琥珀色（amber）（2026-06-24 kako-jun 確定。旧・灰青は廃止）。盲目＋「夢で初めて色を見る」設定とも響く特異性。生成プロンプトに `heterochromia, odd eyes (one jade green eye, one amber eye)` を入れる。
  - **感情アーク（定番）**：せおは「スッキリ」が大事。問いに答えてもらうまでは**モヤモヤ＝赤らめ顔（顔が赤らんでいる）**、疑問が解決すると**スッキリ＝通常顔**でシナリオが終わる、が定番。→ 立ち絵の表情差分は **通常顔／赤らめ顔の2枚が基本**（制作: studio-yokonami #9 赤らめ顔・#6 せお専用LoRA）。スクリプトは「すっきりした」のビートで表情を 赤らめ→通常 に切り替える。
  - **定番と例外**：上記スッキリ締めは**定番＝基本**。ただし **例外はいくつあってもよい（マンネリこそ最大の敵）**（kako-jun 確定）。司会ヴィンチアが最後に追い打ちで再び問いを開く構成（例：avatar__kantia は「すっきりした！」の後にヴィンチアが揺り戻し、せおが再びモヤモヤ＝赤らめ顔寄りで余韻を残す）は**意図的な例外として維持**する。全体テーマ「卒業とは答えを信仰せず、問いを持ち帰ること」（four_act_structure）とも合致。赤らめ→通常の表情切替は各脚本のビートで個別に設計する（定番に縛らない）。
- **ヴィンチア**：司会。人類最初の「知りたい」衝動の人格化。**黒幕ではない**（絶対ルール）。

平均顔（mode collapse）回避のため、住人の何人かに**強い視覚署名**を与える。**元キャラだとバレる手前で止め、特徴だけを中性青年に溶かす**（IP衛生＝art_and_voice 準拠。生成プロンプトに元キャラ名は出さない）。

|---|---|---|
| デカリス（懐疑） | 天才の隈・猫背 | 目の下の濃い隈、やや猫背、気だるく鋭い・不気味だが優しい天才（探偵的＝方法的懐疑と一致） |
| ヒュー（経験） | 三白眼 | 下三白眼、とぼけた半眼、脱力した三枚目（皮肉屋で砕けた経験論と一致） |
| カンティア（認識） | 鋭い眼光 | 目つきが鋭く冷たげだが芯は優しい、規律的・端正（潔癖な認識論と一致） |
| マキヤ（現実） | 後ろ刈り上げ | ツーブロック／アンダーカット（後ろ・サイドを刈り上げ、トップは残す）、現代的でシャープな実務家 |
| オウ（知行） | 閉眼 | 常に目を閉じている（沈着・内省）。眼光系の署名は載らない＝独立署名 |

- せお（主人公）・ヴィンチア（司会）・アリスト・ヘグル・スピノ は今回の署名対象外（せお＝プレーン目＋オッドアイ翡翠/琥珀、他は ToHeart 目パレット路線を維持）。
- 各署名は**眼／髪の上書き**で、同一性（顔の作り・衣装・配色）は各キャラの参考画像／既存設計を維持する。

## 口調・呼称（声の設計・2026-06-24 kako-jun 確定 / #7）

声が被ると「同じ人が喋っている」ように聞こえる（kako-jun 実機指摘: せおが女声＝ボクっ娘、オウとデカリスがどちらも「お前＋俺」で被り）。**各キャラの一人称・せお呼び・語尾(register) を排他にする**。脚本（`content/scripts/`）はこの表に従う。

| キャラ | 一人称 | せお呼び（二人称） | register（語尾・話し方） | 弁別のキモ |
|---|---|---|---|---|
