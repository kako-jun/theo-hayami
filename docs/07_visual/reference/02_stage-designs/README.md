# 舞台デザイン

背景、舞台、画面空間の画像を置く。

シートから切り出した小さい背景は構図参照だけに使う。採用候補は `../BACKGROUND_IMAGE_SPEC.md` に従って単体高解像度で再生成する。

主な対象:

- `shadow-library/` 影の図書館
- `information-layer/` 情報層
- `idea-world/` イデア界
- `holographic-universe/` ホログラフィック宇宙
- `reality/` 現実側

採用判断の観点:

- 読書・対話・消失が起きる場所として使えるか。
- ただの幻想背景でなく、物語上の機能が見えるか。
- name-name 埋め込み時に文字・立ち絵・UIを邪魔しないか。
- 暗さや情報量が強すぎて、本文体験を食っていないか。
