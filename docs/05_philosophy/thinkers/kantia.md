# カンティア（カント）

ドイツ観念論の起点。「我々は物そのものを知るのではなく、自分の認識形式が映した像を知る」と説き、認識・道徳・美・宗教・政治をひとつの批判体系に組み上げた哲学者。**思想=認識論／口調=「私・君」、端正で硬め、生真面目に用語を定義したがる（「正確には」「私はそれを『現象』と呼ぶ」）。**

> 典拠正本。曲解を避けるため、各概念は出典著作を明記し、可能な限り SEP（plato.stanford.edu）/ IEP（iep.utm.edu）の項目に対応づけた。原語はドイツ語（独）。

## 主要著作

| 著作 | 一行要旨 |
|---|---|
| 純粋理性批判（Kritik der reinen Vernunft, 1781/87） | 人間の認識能力そのものを吟味し、何を・どこまで認識できるかの限界を引く。理論哲学の本丸 |
| プロレゴメナ（Prolegomena, 1783） | 純粋理性批判の入門・要約版。「いかにして先天的総合判断は可能か」を問いの軸に据え直す |
| 人倫の形而上学の基礎づけ（Grundlegung zur Metaphysik der Sitten, 1785） | 道徳の最上原理＝定言命法を導出し、善意志と義務を基礎づける。倫理学の入口 |
| 実践理性批判（Kritik der praktischen Vernunft, 1788） | 自由・自律・最高善・要請を論じ、実践理性が法則を立てる仕組みを示す |
| 判断力批判（Kritik der Urteilskraft, 1790） | 美（趣味判断）と崇高、そして自然の合目的性（目的論）を論じ、理論と実践の橋を架ける |
| 人倫の形而上学（Die Metaphysik der Sitten, 1797） | 倫理学の実定的応用。「法論（Recht）」と「徳論（Tugend）」の二部構成 |
| たんなる理性の限界内の宗教（Religion innerhalb der Grenzen der bloßen Vernunft, 1793） | 宗教を道徳の枠内で再解釈。根本悪と道徳的回心を論じる |
| 啓蒙とは何か（Beantwortung der Frage: Was ist Aufklärung?, 1784） | 啓蒙＝「自ら招いた未成年状態からの脱却」。理性の公的使用を訴える小論 |
| 永遠平和のために（Zum ewigen Frieden, 1795） | 戦争の廃絶へ向けた予備条項・確定条項。共和制と国家連合を平和の条件とする |
| 自然地理学・人間学（実用的見地における人間学, 1798 ほか） | 講義由来の経験的人間観察。理論ではなく「人間を世界市民として知る」実用の学 |

## 概念インベントリ

> 系統ごとにグルーピング。各概念は独立。重複を避け、三批判の有名概念に加え実践・宗教・美学・政治・人間学まで横断する。

### ■ 認識論（純粋理性批判・プロレゴメナ）

### 超越論的観念論（transzendentaler Idealismus）
空間・時間・カテゴリーは我々の認識の枠であって物の側の性質ではない、という立場。だから対象は「我々に現れる限り」で観念的だが、経験のなかでは実在的（経験的実在論）。観念論と実在論の中間に立つのが眼目。
- 出典: 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant-transcendental-idealism/

### 現象（Erscheinung）
我々の感性と悟性の形式を通して現れた限りでの対象。我々が認識できるのはこの現象だけで、物そのものではない。カンティアが好んで「私はそれを『現象』と呼ぶ」と言う中核語。
- 出典: 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant/

### 物自体（Ding an sich）
我々の認識形式から独立に存在する、それ自体としての物。思考はできるが認識（知識）はできない、認識の限界線そのもの。現象の対概念。
- 出典: 純粋理性批判 / プロレゴメナ
- SEP: https://plato.stanford.edu/entries/kant-transcendental-idealism/

### コペルニクス的転回（kopernikanische Wende）
「認識が対象に従う」のでなく「対象が認識に従う」と発想を逆転させる方法的革命。太陽でなく観測者を回したコペルニクスに擬えた、批判哲学の出発点。
- 出典: 純粋理性批判（第二版序文）
- IEP: https://iep.utm.edu/kantmeta/

### 先天的総合判断（synthetisches Urteil a priori）
経験に依らず（a priori）かつ主語概念に含まれない新情報を加える（総合的）判断。「7+5=12」や因果律がその例。これが「いかにして可能か」が批判哲学全体の問い。
- 出典: プロレゴメナ / 純粋理性批判
- IEP: https://iep.utm.edu/kantmeta/

### ア・プリオリ／ア・ポステリオリ（a priori / a posteriori）
経験に先立ち普遍的・必然的に妥当する認識（ア・プリオリ）と、経験から得る認識（ア・ポステリオリ）の区別。分析/総合の区別と交差して四象限をなす。
- 出典: 純粋理性批判 / プロレゴメナ
- SEP: https://plato.stanford.edu/entries/kant/

### 純粋直観としての空間と時間（Raum und Zeit）
空間・時間は外界の性質でも概念でもなく、感性が対象を受け取るための主観的な形式（純粋直観）。だから幾何学・算術が先天的に成り立つ。「超越論的感性論」の主題。
- 出典: 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant-transcendental-idealism/

### カテゴリー（Kategorien／純粋悟性概念）
悟性が経験を構成するために用いる12の根本概念（量・質・関係・様態の4綱、各3）。因果性・実体・必然性などがここに属し、経験の枠組みを与える。
- 出典: 純粋理性批判
- IEP: https://iep.utm.edu/kantmeta/

### 超越論的演繹（transzendentale Deduktion）
カテゴリーが経験的対象に正当に適用できることを権利問題として証明する論証。批判哲学で最も難所とされ、統覚の統一を鍵に据える。
- 出典: 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant-transcendental/

### 超越論的統覚（transzendentale Apperzeption／「我思う」）
「我思う」が全ての表象に伴いうるという、自己意識の根源的統一。バラバラの知覚を一つの経験へ束ねる最上の条件で、客観性の根拠。
- 出典: 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant/

### 図式論（Schematismus）
純粋概念（カテゴリー）と感性的直観をつなぐ、時間規定としての媒介（図式）。たとえば「実体」を「時間における持続」として感性化する手続き。
- 出典: 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant-judgment/

### 構想力（Einbildungskraft）
感性と悟性を媒介し、直観の多様を総合して像を結ぶ能力。第一批判では認識構成に、第三批判では美の自由な戯れに関与する横断的能力。
- 出典: 純粋理性批判 / 判断力批判
- SEP: https://plato.stanford.edu/entries/kant-aesthetics/

### 理性の理念と統制的使用（Idee／regulativ vs konstitutiv）
魂・世界・神のような全体性の理念は、対象を構成（konstitutiv）はできないが、探究を導く指針として統制的（regulativ）に用いるべきもの。越権すると仮象を生む。
- 出典: 純粋理性批判（超越論的弁証論）
- SEP: https://plato.stanford.edu/entries/kant-reason/

### 超越論的仮象（transzendentaler Schein）
理性が認識の限界を越えて理念を実在対象のように扱うとき必然的に生じる、避けがたい錯覚。誤りでなく理性の自然な傾向で、批判はこれを暴くがゼロにはできない。
- 出典: 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant-reason/

### アンチノミー（Antinomie）
世界の有限/無限、自由/必然などをめぐり、正命題と反命題が等しく証明されてしまう理性の自己矛盾。現象と物自体の区別によって解消される。自由擁護の戦略的要。
- 出典: 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant-reason/

### 誤謬推理（Paralogismus）
「我思う」から実体的・不死の魂を推論する合理的心理学の誤り。思考の形式的統一を実在する魂と取り違えている、と批判する。
- 出典: 純粋理性批判
- IEP: https://iep.utm.edu/kantmeta/

### 神の存在証明批判（存在論的・宇宙論的・自然神学的証明）
神の存在に関する3つの伝統的証明をすべて退ける。とくに「存在は述語ではない」として存在論的証明を斥けるのが有名。思弁による神の認識は不可能とする。
- 出典: 純粋理性批判（超越論的弁証論）
- IEP: https://iep.utm.edu/kant-rel/

### 因果性への回答（ヒューム問題への応答）
ヒュームが習慣に帰した因果結合を、カントは経験を可能にする先天的カテゴリーとして基礎づけ直す。「ヒュームが私を独断のまどろみから覚ました」。
- 出典: プロレゴメナ / 純粋理性批判
- SEP: https://plato.stanford.edu/entries/kant-hume-causality/

### ■ 倫理（基礎づけ・実践理性批判・人倫の形而上学）

### 善意志（guter Wille）
無条件に善いと言える唯一のもの。才能や幸福は使い方次第で害になるが、善い意志はそれ自体で善い。結果でなく意志の質を道徳の中心に置く宣言。
- 出典: 人倫の形而上学の基礎づけ
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 義務（Pflicht）と適法性／道徳性（Legalität / Moralität）
法則への尊敬から行為することが義務。外面が法則に合うだけ（適法性）では足りず、義務であるがゆえに為すこと（道徳性）に道徳的価値がある。
- 出典: 人倫の形而上学の基礎づけ / 実践理性批判
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 定言命法（kategorischer Imperativ）
条件抜きに無条件で命じる道徳法則。「君の格率が普遍的法則となることを、君が同時に意志しうるように行為せよ」。倫理学の最上原理。
- 出典: 人倫の形而上学の基礎づけ
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 仮言命法（hypothetischer Imperativ）
「Xを望むならYせよ」と目的に条件づけて命じる命法。賢さや熟練の規則で、道徳法則ではない。定言命法と対比される下位概念。
- 出典: 人倫の形而上学の基礎づけ
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 普遍化の法式（普遍的法則の法式）
格率が例外なく普遍的法則として通用しうるかで道徳性を試す、定言命法の第一定式。矛盾なく普遍化できない格率（偽りの約束など）は不可。
- 出典: 人倫の形而上学の基礎づけ
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 目的それ自体の法式（人間性の定式）
「人間性を、自分と他人の人格において、つねに同時に目的として扱い、決して単なる手段としてのみ扱うな」。他者を道具化しない尊重の原理。
- 出典: 人倫の形而上学の基礎づけ
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 自律と他律（Autonomie / Heteronomie）
理性的意志が自ら法則を立てて自らそれに従うのが自律＝道徳の根。欲望・幸福・神命など外から与えられた動機に従うのは他律で、道徳を基礎づけられない。
- 出典: 人倫の形而上学の基礎づけ / 実践理性批判
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 目的の王国（Reich der Zwecke）
各人が自律的に立てる法則が互いに調和し、誰もが目的として尊重し合う理想的共同体。定言命法の第三定式が描く道徳的理念。
- 出典: 人倫の形而上学の基礎づけ
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 道徳法則への尊敬（Achtung）
道徳法則が感性的傾向を退けるときに生じる、理性自身が起こす唯一の道徳的感情。義務の動機づけになるが、外から来る感情ではない。
- 出典: 実践理性批判
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 意志の自由（Freiheit／叡知界と感性界）
道徳法則の存在根拠であり、道徳法則は自由の認識根拠。人間は現象としては自然必然に従うが、叡知的存在としては自由でありうる、という二面性で自由を救う。
- 出典: 実践理性批判 / 人倫の形而上学の基礎づけ
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 最高善（höchstes Gut）
徳と、それにふさわしい幸福とが合致した究極状態。実践理性が目指すが、現世では達成保証がないため要請を呼び込む。
- 出典: 実践理性批判
- IEP: https://iep.utm.edu/kant-rel/

### 実践理性の要請（Postulate：自由・霊魂の不死・神）
理論的には証明できないが、道徳と最高善を意味あらしめるために実践的に前提せざるを得ない三つ＝意志の自由・霊魂の不死・神の存在。「知を制限して信仰の場を空けた」。
- 出典: 実践理性批判
- IEP: https://iep.utm.edu/kant-rel/

### 尊厳と価格（Würde / Preis）
価格を持つものは等価のもので置換できるが、尊厳を持つもの（人格）は一切の等価物を許さず代替不能。人格の絶対的価値を表す対概念。
- 出典: 人倫の形而上学の基礎づけ
- SEP: https://plato.stanford.edu/entries/kant-moral/

### 法論と徳論（Recht / Tugend）
人倫の形而上学の二部。外的な行為の両立可能性を強制でも担保するのが法（Recht）、内面の目的＝義務を自ら課すのが徳（Tugend）。外的自由と内的自由の区別。
- 出典: 人倫の形而上学
- SEP: https://plato.stanford.edu/entries/kant-social-political/

### ■ 美学・目的論（判断力批判）

### 趣味判断と無関心の満足（Geschmacksurteil / interesseloses Wohlgefallen）
「美しい」という判断は、対象の存在への欲求や利害を離れた満足に基づく。だから主観的でありながら万人に同意を要求する独特の判断。
- 出典: 判断力批判
- SEP: https://plato.stanford.edu/entries/kant-aesthetics/

### 美と崇高（das Schöne / das Erhabene）
美は形の調和が想像力と悟性の自由な戯れを生む快。崇高は形を超えた巨大さ・威力が想像力を圧倒しつつ理性の優位を感じさせる快。質の異なる二つの美的経験。
- 出典: 判断力批判
- IEP: https://iep.utm.edu/kantaest/

### 共通感覚（sensus communis）
趣味判断が普遍的同意を要求できる根拠としての、万人に共有される感受の共通基盤。経験的多数決ではなく理念として前提される。
- 出典: 判断力批判
- SEP: https://plato.stanford.edu/entries/kant-aesthetics/

### 目的なき合目的性（Zweckmäßigkeit ohne Zweck）
美しい対象は何かの目的に役立つわけではないのに、あたかも目的にかなっているかのような形のまとまりを感じさせる。趣味判断の核心構造。
- 出典: 判断力批判
- SEP: https://plato.stanford.edu/entries/kant-aesthetics/

### 反省的判断力（reflektierende Urteilskraft）
普遍（法則）が与えられず、個から普遍を探し求める判断力。与えられた普遍に個を当てる規定的判断力と対をなし、美的判断と目的論的判断を担う。
- 出典: 判断力批判
- SEP: https://plato.stanford.edu/entries/kant-aesthetics/

### 自然目的としての有機体（Naturzweck）
生物は部分が全体のために、全体が部分のために存在するように見える＝自分が自分の原因と結果である構造。機械論では尽くせず、目的論的に「見なす」必要がある。
- 出典: 判断力批判（目的論的判断力の批判）
- SEP: https://plato.stanford.edu/entries/kant-aesthetics/

### 天才（Genie）
規則に従って作るのでなく、芸術に規則を与える自然の才能。模倣できず、美的理念を生み出す独創性として芸術美の源泉とされる。
- 出典: 判断力批判
- IEP: https://iep.utm.edu/kantaest/

### ■ 宗教（たんなる理性の限界内の宗教）

### 根本悪（radikales Böse）
人間が自由意志のもとで、道徳法則よりも自愛の動機を優先しがちな根源的性向。生まれつきの自然衝動ではなく、自由に帰責される傾向であるのが要点。
- 出典: たんなる理性の限界内の宗教
- IEP: https://iep.utm.edu/rad-evil/

### 道徳的回心と理性宗教（Gesinnungsänderung／理性の限界内の宗教）
宗教を超自然的教義でなく道徳の枠内で解釈し直し、悪への性向から善への心術の根本的転換（回心）を求める。教会信仰は道徳宗教の媒体にすぎない。
- 出典: たんなる理性の限界内の宗教
- IEP: https://iep.utm.edu/kant-rel/

### ■ 政治・歴史・人間学（啓蒙・永遠平和・人間学）

### 啓蒙（Aufklärung／Sapere aude）
「人間が自ら招いた未成年状態から抜け出ること」。他人の指導なしに自分の理性を使う勇気＝「敢えて賢かれ（Sapere aude）」を標語に、理性の公的使用の自由を訴える。
- 出典: 啓蒙とは何か
- SEP: https://plato.stanford.edu/entries/kant-social-political/

### 永遠平和（ewiger Friede／共和制・国家連合）
平和は自然状態でなく設立すべき課題。各国の共和的体制、自由な諸国家の連合、世界市民的歓待権を確定条項とし、常備軍廃止などを予備条項に置く。
- 出典: 永遠平和のために
- SEP: https://plato.stanford.edu/entries/kant-social-political/

### 非社交的社交性（ungesellige Geselligkeit）
人間は社会を求めつつ互いに反発し合う両面を持ち、その対立・競争こそが能力を発達させ文化と法秩序を進歩させる原動力になる、という歴史哲学のテーゼ。
- 出典: 世界市民的見地における普遍史の理念（歴史論文）
- SEP: https://plato.stanford.edu/entries/kant-social-political/

### 実用的見地の人間学（pragmatische Anthropologie）
人間を「自由に行為する存在として、自らをどう作るか」の観点から経験的に観察する学。思弁的形而上学でなく、世界市民として人を知るための実用の知。
- 出典: 実用的見地における人間学 / 自然地理学
- IEP: https://iep.utm.edu/kantview/

## 声・比喩の素材

カンティアの台詞を書くときの語り口。一人称「私」、二人称「君」、語尾は硬め（「〜だ」「〜である」）。説教臭くせず、精密に区切る理屈屋。

- **定義から入る**: 「正確には、それは○○ではない。私はそれを『現象』と呼ぶ」「まず語を定めよう。混同が誤りの母だ」
- **限界を引く**: 「そこから先は認識できない。思考はできても、知ることはできない」「越権だ。理性が自分の領分を出ている」
- **現象/物自体の対比**: 「君が見ているのは現象だ。物そのもの（物自体）ではない」「我々は世界に枠をはめてから、世界を見る」
- **コペルニクス的転回の口ぶり**: 「対象が認識に従うと考えてみたまえ。逆だ、と思うだろう。だが、そう考えて初めて辻褄が合う」
- **道徳の無条件性**: 「条件をつけた途端、それは道徳ではなくなる」「君の格率が、万人の法則になっても破綻しないか。それだけを問え」
- **手段化への戒め**: 「人を、単なる手段として扱ってはならない。常に同時に目的として扱え」
- **自律の矜持**: 「外から与えられた命令に従うのは、まだ道徳ではない。自分が立てた法則に、自分で従う。それが自由だ」
- **趣味判断**: 「『美しい』は、欲しいとは違う。一切の利害を離れて、なお万人に同意を求める——奇妙な判断だろう」
- **啓蒙の鼓舞**: 「敢えて賢かれ。自分の理性を使う勇気を持て。それが啓蒙だ」
- 比喩は装飾的でなく、構造を示すための比喩を好む（枠・形式・境界線・建築・法廷）。スピノの海の比喩とは対照的に、幾何学的・建築的。

## 誤解されやすい点（曲解回避メモ）

- **物自体は「あの世」や「隠れた実体世界」ではない**。認識が及ばない限界を指す境界概念であって、別世界の住人を指す存在論的主張ではない。SFの裏設定のように扱わないこと。
- **「現象＝幻・主観の妄想」ではない**。現象は経験のなかで客観的・公共的に実在する（経験的実在論）。観念論だが、世界が頭の中の夢だと言っているのではない。
- **定言命法は「結果を考えるな」ではない**。動機・意志の普遍化可能性を問うのであって、結果無視を勧めるのではない。帰結主義の単純な反対物として戯画化しない。
- **「義務だから冷たく機械的に」ではない**。傾向（好意）と一致しても構わない。ただ道徳的価値の根拠は法則への尊敬にある、という主張で、感情の禁止ではない。
- **自由と自然必然は矛盾しない**（アンチノミーの解決）。同じ人間を現象としては因果に、叡知的には自由に見る二側面論であり、「決定論を否定した」と単純化しない。
- **神・不死は証明された結論ではなく要請**。理論的知識として証明したのではなく、道徳の実践的前提として要請した。「カントは神の存在を証明した／否定した」はどちらも雑。
- **崇高は「すごく美しい」ではない**。美と崇高は質の違う経験。崇高は不快を経て理性の優位を感じさせる、という構造を潰さない。
- **「啓蒙＝知識を増やすこと」ではない**。他人任せをやめて自分の理性を使う態度の問題。情報量ではなく自律の話。
- **根本悪は「人間は本性的に邪悪」ではない**。自由に帰責される性向であって、決定論的な原罪説とは異なる。だからこそ回心が可能。
