# アリスト（アリストテレス）

古代ギリシアの哲学者アリストテレス（前384–322）。プラトンの弟子でありながら、観察と分類によって世界を理解しようとした「万学の祖」。倫理・論理・自然・政治・詩学まで、ほぼ全領域に体系を築いた。

**『せおはやみ』での役割** — 思想系統は「**分類と観察**」。空理空論より、まず目の前のものを観て、種類ごとに分けて並べることから始める博物学者。口調は穏やかで有能な整理魔（一人称「私」／せお呼び「きみ」／「〜だね」「まず分けて見よう」）。他の登場人物が抽象や情熱で押す中、アリストは静かに「では、これを分けてみよう」と土台を据える。形而上の高みも語れるが、その語り口はいつも「観察 → 区別 → 名づけ」の手順を踏む。

---

## 主要著作

- **ニコマコス倫理学** — 人間の善（幸福）と徳を、習慣と中庸から論じる倫理学の正典。
- **エウデモス倫理学** — ニコマコス倫理学と多くを共有する、もう一つの倫理学講義。
- **形而上学** — 「有を有として」研究する第一哲学。実体・原因・不動の動者。
- **自然学（フュシカ）** — 自然・運動・原因・場所・時間など、変化する世界の原理。
- **魂について（デ・アニマ）** — 魂を身体の形相として捉え、生命・感覚・知性を論じる。
- **政治学** — ポリス（国家）を人間の自然な共同体として分析、国制を分類する。
- **詩学（ポエティカ）** — 悲劇の構造と効果を分析した、最初期の体系的文芸論。
- **カテゴリー論** — 存在を述語の十類型に分ける、オルガノンの入口。
- **分析論前書・後書** — 三段論法と論証科学の理論（オルガノン中核）。
- **トピカ／命題論（オルガノン）** — 弁証術・通念・命題と対当の論理。
- **弁論術（レトリケー）** — 説得の手段を観察する技術。弁証術の対をなす。
- **動物誌・動物部分論・動物発生論** — 動物の観察・分類・目的論的説明（生物学）。
- **天体論（デ・カエロ）** — 天界と元素、第五元素（エーテル）の円運動。

## 概念インベントリ

> 系統別。各概念は独立して脚本の引用素材に使える。出典著作と、確認に用いた二次文献（SEP/IEP）を併記。

### A. 論理・認識（オルガノン）

### カテゴリー（κατηγορίαι / katēgoriai）
存在について語られうる述語の最高十類型＝実体・量・質・関係・場所・時間・位置・所有・能動・受動。世界を語る前に、まず「何について」の種類を分ける枠組み。
出典: カテゴリー論／形而上学Δ
参照: https://plato.stanford.edu/entries/aristotle-categories/

### 第一実体と第二実体（prōtē / deutera ousia）
個物（このソクラテス、この馬）が第一実体で最も真に「ある」もの。種・類（人間、動物）は第二実体。普遍より個が存在の基底。
出典: カテゴリー論
参照: https://plato.stanford.edu/entries/aristotle-categories/

### 三段論法（συλλογισμός / syllogismos）
「いくつかのことが置かれると、それとは別のことが必然的に帰結する」推論形式。中名辞（middle term）を介して二前提から結論を導く、演繹の骨格。
出典: 分析論前書
参照: https://plato.stanford.edu/entries/aristotle-logic/

### 論証と学的知（ἀπόδειξις / apodeixis・ἐπιστήμη / epistēmē）
論証とは「知識を生む三段論法」。真・第一・無媒介の前提から、原因を通じて必然的結論を導く。これが成り立つときだけ学的知（episteme）が成立する。
出典: 分析論後書
参照: https://plato.stanford.edu/entries/aristotle-logic/

### 第一原理と知性的把握（ἀρχαί / archai・νοῦς / nous）
論証はどこかで止まる。論証できない出発点＝第一原理は、知性（nous）が直接に把握する。学の鎖は無限後退せず、最初の杭で支えられる。
出典: 分析論後書 II.19
参照: https://plato.stanford.edu/entries/aristotle-logic/

### 帰納（ἐπαγωγή / epagōgē）
個別から普遍へ進む推論。多くの事例の観察を重ね、共通する普遍を浮かび上がらせる。第一原理を得る道でもある。観察者アリストの方法そのもの。
出典: 分析論後書／トピカ
参照: https://plato.stanford.edu/entries/aristotle-logic/

### 弁証術と通念（διαλεκτική / dialektikē・ἔνδοξα / endoxa）
学の確実な前提ではなく「広く受け入れられた見解（通念）」から出発し、問答で吟味する技術。論破と探究の両方に使える、思考の対戦稽古。
出典: トピカ／ソフィスト的論駁
参照: https://plato.stanford.edu/entries/aristotle-logic/

### 述語可能項（predicables）
ある主語に対し述語が立つ仕方の五種＝類（genus）・種差（differentia）・種（species）・固有性（property）・付帯性（accident）。定義を組む部品の分類。
出典: トピカ
参照: https://plato.stanford.edu/entries/aristotle-logic/

### 矛盾律（law of non-contradiction）
「同じものが、同じ点で、同時に、あるものに属しかつ属さないことは不可能」。あらゆる論証が前提とする、否定できない最も確実な原理。
出典: 形而上学Γ（IV）
参照: https://plato.stanford.edu/entries/aristotle-metaphysics/

### B. 形而上学

### 有を有として（τὸ ὂν ᾗ ὄν / on hēi on）
個々の性質ではなく「ある」というそのこと自体を研究する第一哲学。物理学が動く物を、数学が量を扱うのに対し、これは存在を存在として問う。
出典: 形而上学Γ・Ε
参照: https://plato.stanford.edu/entries/aristotle-metaphysics/

### 実体（οὐσία / ousia）
「他のものがそれについて語られるが、それ自身は他について語られないもの」＝最も基底的に存在するもの。すべてのカテゴリーの中心。
出典: 形而上学Ζ・Η／カテゴリー論
参照: https://plato.stanford.edu/entries/aristotle-metaphysics/

### 質料形相論（ヒュロモルフィズム / hylomorphism）
個物は質料（素材）と形相（構造・本質）の合成体。青銅（質料）と像の形（形相）が一つの彫像になるように、両者の結合として実体が成る。
出典: 形而上学Ζ・Η／自然学
参照: https://plato.stanford.edu/entries/aristotle-metaphysics/

### 可能態と現実態（δύναμις / dunamis・ἐνέργεια / energeia・ἐντελέχεια / entelecheia）
種子は樹になりうる「可能態」、現に樹であるのが「現実態」、完成して目的に達した状態が「完全現実態（エンテレケイア）」。変化と存在の根本対。現実態が存在・時間・定義において先立つ。
出典: 形而上学Θ／自然学
参照: https://plato.stanford.edu/entries/aristotle-metaphysics/

### 四原因（αἰτίαι / aitiai）
「なぜ」への四つの答え＝質料因（何でできているか）・形相因（何であるか／本質）・作用因（何が生んだか）・目的因（何のためか）。説明とは四方向から原因を挙げること。
出典: 自然学II／形而上学
参照: https://plato.stanford.edu/entries/aristotle-causality/

### 本質（τὸ τί ἦν εἶναι / to ti ēn einai）
「それがそれであるところのもの」＝定義によって言い表される、そのものの核。直訳は「あったところのものであること」。付帯性と違い、それが失われればもう同じものではない。
出典: 形而上学Ζ
参照: https://plato.stanford.edu/entries/aristotle-metaphysics/

### 不動の動者（πρῶτον κινοῦν ἀκίνητον / unmoved mover）
自らは動かずに、すべての運動の究極原因となる永遠・非物質の存在。憧れの対象として（愛されることで）天を動かす。アリストの神＝純粋な思考の思考。
出典: 形而上学Λ（XII）／自然学VIII
参照: https://plato.stanford.edu/entries/aristotle-metaphysics/

### 「ある」は多様に語られる（πολλαχῶς λέγεται / pollachōs legetai）
「ある」は一義ではない。実体としてある・量としてある・関係としてある…と様々に言われ、しかしすべて実体を中心に（pros hen）束ねられる。
出典: 形而上学Γ・Δ
参照: https://plato.stanford.edu/entries/aristotle-metaphysics/

### C. 自然学・天体論

### 自然（φύσις / phusis）
「変化と静止の内的原理」。人工物が外から作られるのに対し、自然物は自分の内に動き・育つ原理を持つ。石は落ち、種は芽吹く——内側から。
出典: 自然学II
参照: https://plato.stanford.edu/entries/aristotle-natphil/

### 運動・変化（κίνησις / kinēsis）
「可能態にあるものの、可能態である限りでの現実化」。実体（生成消滅）・質・量・場所の四領域で起こる、潜在から顕在への移行過程。
出典: 自然学III
参照: https://plato.stanford.edu/entries/aristotle-natphil/

### 目的論（τέλος / telos・teleology）
自然の過程は目的へ向かう。「自然は何ものも無駄に作らない」。雨が穀物のために降るかは慎重に論じるが、生物の器官は機能（目的）から説明される。
出典: 自然学II／動物部分論
参照: https://plato.stanford.edu/entries/aristotle-natphil/

### 場所と自然な場所（τόπος / topos・natural place）
場所とは、囲む物体の内側の境界。各元素には本来あるべき「自然な場所」があり（重いものは中心へ、軽いものは上へ）、そこへ向かうのが自然運動。
出典: 自然学IV／天体論
参照: https://plato.stanford.edu/entries/aristotle-natphil/

### 時間（χρόνος / chronos）
「先と後に関する運動の数（測り）」。時間は運動から独立しては存在せず、変化を数えることで現れる。動きを測る心が、時間を時間にする。
出典: 自然学IV
参照: https://plato.stanford.edu/entries/aristotle-natphil/

### 無限（ἄπειρον / apeiron）
無限は「現実態としては」存在しない。数えても尽きない・分割し続けられるという「可能態としての無限」だけが認められる。完結した無限大は否定。
出典: 自然学III
参照: https://plato.stanford.edu/entries/aristotle-natphil/

### 連続（συνεχές / syneches）
大きさ・運動・時間は連続量であり、点や瞬間の寄せ集めではない。無限に分割できるが、分割の部分から組み立てられるのではない。ゼノン論駁の土台。
出典: 自然学VI
参照: https://plato.stanford.edu/entries/aristotle-natphil/

### 四元素と第五元素（elements・αἰθήρ / aether）
地・水・火・風は月下の世界で直線運動する。月より上の天界は不生不滅の第五元素エーテルから成り、円運動する。世界は二つの異なる物理に分かれる。
出典: 天体論
参照: https://plato.stanford.edu/entries/aristotle-natphil/

### D. 魂について（デ・アニマ）

### 魂（ψυχή / psuchē）
魂は身体から切り離せる霊ではなく、「生命を可能態として持つ自然的身体の第一現実態（形相）」。生きていることそのものの原理。蝋とその刻印のように、身体と一体。
出典: 魂についてII
参照: https://plato.stanford.edu/entries/aristotle-psychology/

### 魂の三能力（栄養・感覚・理性）
魂は段階をなす。植物は栄養能力（成長・生殖）のみ、動物はそれに感覚・運動を加え、人間はさらに理性を持つ。上の段階は下を含む入れ子構造。
出典: 魂についてII
参照: https://plato.stanford.edu/entries/aristotle-psychology/

### 感覚（αἴσθησις / aisthēsis）
感覚とは「質料抜きに感覚的形相を受け取ること」。蝋が金属抜きに印章の形を受けるように、目は色を、耳は音の形を受容する。知の出発点。
出典: 魂についてII・III
参照: https://plato.stanford.edu/entries/aristotle-psychology/

### 共通感覚（κοινὴ αἴσθησις / koinē aisthēsis）
各個別感覚を束ね、運動・形・数のような複数感覚に共通する性質を捉え、「これは甘くて白い」と統合判別する中枢的能力。
出典: 魂についてIII
参照: https://plato.stanford.edu/entries/aristotle-psychology/

### 想像（φαντασία / phantasia）
感覚が去った後も像を保ち、感覚と思考の間を媒介する能力。記憶・夢・欲求の基盤であり、対象が不在でもその姿を心に立ち上げる。
出典: 魂についてIII
参照: https://plato.stanford.edu/entries/aristotle-psychology/

### 能動知性と受動知性（νοῦς ποιητικός / passive・active intellect）
知性は二相を持つ。あらゆる可知的形相になりうる受動知性と、それを現実に知らしめる能動知性。後者は「分離し・不受動・不滅」と言われ、解釈史上の難所。
出典: 魂についてIII.5
参照: https://plato.stanford.edu/entries/aristotle-psychology/

### 欲求（ὄρεξις / orexis）
動物の目的的運動を起こす力。理性（または感覚像）が目標を示し、欲求がそれへ向かわせる。知だけでは動かない——欲求が体を動かす。
出典: 魂についてIII／動物運動論
参照: https://plato.stanford.edu/entries/aristotle-psychology/

### E. 倫理学

### エウダイモニア（εὐδαιμονία / eudaimonia）
「幸福」と訳されるが、瞬間の快でなく「徳に基づく魂の活動」としての善き生・開花（flourishing）。すべての行為が最終的に目指す、それ自体のために望まれる最高善。
出典: ニコマコス倫理学I・X／エウデモス倫理学
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 機能論証（ἔργον / ergon argument）
人間に固有の機能（ergon）は「理性的部分に従った魂の活動」。良い竪琴奏者がよく弾くように、人間の善はその固有の機能を徳をもって発揮すること。
出典: ニコマコス倫理学I.7
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 徳（ἀρετή / aretē）
卓越性・うまくあること。性格の徳（勇気・節制など、習慣で育つ）と知性の徳（知恵・思慮、教育で育つ）に分かれる。感情と行為を適切に向ける安定した性向。
出典: ニコマコス倫理学II
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 習慣づけと性向（ἕξις / hexis・ἐθισμός / ethismos）
性格の徳は生まれつきでなく反復で身につく。「正しい行為を重ねて正しい人になる」。一度きりでなく、繰り返しが安定した性向（hexis）を形づくる。
出典: ニコマコス倫理学II.1
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 中庸（μεσότης / mesotēs / doctrine of the mean）
徳は二つの悪徳（過剰と不足）の間の中間にある。勇気は無謀と臆病の間。ただし算術的中点でなく「われわれにとっての中（状況に応じた適切さ）」。
出典: ニコマコス倫理学II
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 思慮（φρόνησις / phronēsis）
個別状況で「何をなすべきか」をよく熟慮する実践的知恵。一般原則を具体に適用し、性格の徳を正しく働かせる舵取り。賢者（phronimos）が基準。
出典: ニコマコス倫理学VI
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### アクラシア（ἀκρασία / akrasia）
意志の弱さ・抑制のなさ。何が善いか分かっていながら、欲望に負けて反することをしてしまう状態。知が行為に届かない「知っているのに、できない」の分析。
出典: ニコマコス倫理学VII
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 自発的／非自発的（ἑκούσιον / hekousion・ἀκούσιον）
行為が内的原理から為され、状況を知っているとき自発的。外的強制や無知によるなら非自発的。賞賛・非難・責任の帰属はこの区別に立つ。
出典: ニコマコス倫理学III.1
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 友愛（φιλία / philia）
善き生に不可欠な相互の好意。有用・快楽・徳の三種があり、互いの徳を認め合う「徳の友愛」が完全。友は「もう一人の自分」。
出典: ニコマコス倫理学VIII・IX
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 観想（θεωρία / theōria）
理性の最高機能を働かせる純粋な知的活動。永遠の真理を眺めること。最も自足的・持続的で、神の活動に最も近い、人間の最も幸福な生。
出典: ニコマコス倫理学X
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### 配分的正義（distributive justice）
利益や名誉を「各人の価値（功績）に比例して」分ける正義。等しい者には等しく、等しくない者には不等に。匡正的正義（取引の是正）と対をなす。
出典: ニコマコス倫理学V
参照: https://plato.stanford.edu/entries/aristotle-ethics/

### F. 政治学

### 政治的動物（ζῷον πολιτικόν / zōon politikon）
人間は本性上、ポリス（国家共同体）で生きる動物。言葉（ロゴス）を持ち、善悪・正不正を論じ合える点で、群れる蜂や家畜とは違う社会性を持つ。
出典: 政治学I
参照: https://plato.stanford.edu/entries/aristotle-politics/

### ポリス（πόλις / polis）
家・村が積み上がった、自足を達成する自然な共同体。「生きるために生じ、善く生きるために存在する」。全体が部分（個人）に先立つ。
出典: 政治学I・III
参照: https://plato.stanford.edu/entries/aristotle-politics/

### 国制の六分類（πολιτεία / politeia）
支配者の数（一人・少数・多数）×公共の利益を目指すか否か、で六分類。正しい型＝王制・貴族制・国制（polity）、堕落した型＝僭主制・寡頭制・民主制。
出典: 政治学III
参照: https://plato.stanford.edu/entries/aristotle-politics/

### 中間者の政体（polity・middle class）
理論的最善でなく「多くの国に実現可能な最善」。中間層が主体となる混合政体（polity）が、極端な寡頭・民主に走らず最も安定する。
出典: 政治学IV
参照: https://plato.stanford.edu/entries/aristotle-politics/

### 家政と「生来の奴隷」（οἶκος / oikos・natural slavery）
ポリスの最小単位は家（主人・妻・子・奴隷）。アリストは熟慮能力を欠く者の「生来の奴隷」を論じたが、これは現代から見て最も批判される議論（→誤解されやすい点）。
出典: 政治学I
参照: https://plato.stanford.edu/entries/aristotle-politics/

### G. 詩学・弁論術

### ミメーシス（μίμησις / mimēsis）
模倣・再現。詩や悲劇は現実を「模倣」する。だが単なる写しでなく、ありうべき行為を秩序立てて再現することで、世界を理解可能にする創作行為。
出典: 詩学
参照: https://iep.utm.edu/aristotle/

### 筋（μῦθος / mythos）と悲劇の定義
悲劇は「真剣で完結した、一定の大きさを持つ行為の模倣」。その魂は性格でなく筋（出来事の組み立て）。始め・中・終わりを持つ、因果で結ばれた行為の構成が核。
出典: 詩学
参照: https://iep.utm.edu/aristotle/

### カタルシス（κάθαρσις / katharsis）
悲劇が憐れみ（pity）と恐れ（fear）を呼び起こし、それらの感情を浄化・解放する効果。観客の心を揺さぶり、整える。解釈は浄化説・浄罪説など諸説。
出典: 詩学
参照: https://iep.utm.edu/aristotle/

### ハマルティア（ἁμαρτία / hamartia）
悲劇の主人公が陥る「過ち・的外れ」。道徳的悪というより判断・認識の誤り。悪人でも完全な善人でもない者が、過ちゆえに転落するとき最も悲劇的。
出典: 詩学
参照: https://iep.utm.edu/aristotle/

### 逆転と認知（περιπέτεια / peripeteia・ἀναγνώρισις / anagnōrisis）
逆転＝事態が反対方向へ急転すること。認知＝無知から知への転換（誰が誰かが分かる瞬間）。両者が同時に起こるとき悲劇の効果は最高。
出典: 詩学
参照: https://iep.utm.edu/aristotle/

### 詩と歴史（普遍と個別）
歴史は「起こったこと（個別）」を語り、詩は「起こりうること（普遍）」を語る。ゆえに詩は歴史より哲学的で、より高い真理に触れる。
出典: 詩学9
参照: https://iep.utm.edu/aristotle/

### 説得の三様式（ἦθος / ethos・πάθος / pathos・λόγος / logos）
説得は三つの源から成る。語り手の人柄への信頼（ethos）、聴き手の感情への訴え（pathos）、論証そのものの筋（logos）。弁論術＝説得手段を観察する能力。
出典: 弁論術I
参照: https://plato.stanford.edu/entries/aristotle-rhetoric/

### エンテュメーマと弁論の三類型（ἐνθύμημα / enthymeme）
エンテュメーマは「弁論的三段論法」——通念から出発し前提を省く、聴衆向けの推論。弁論は審議（未来・利害）・法廷（過去・正不正）・演示（現在・称賛非難）の三類型に分かれる。
出典: 弁論術I
参照: https://plato.stanford.edu/entries/aristotle-rhetoric/

### H. 生物学（動物誌・部分論・発生論）

### 自然の階梯（scala naturae）
生物は単純なものから複雑なものへ連続的に並ぶ。植物→下等動物→高等動物→人間と、魂の能力が積み増される梯子状の秩序。
出典: 動物誌／魂について
参照: https://plato.stanford.edu/entries/aristotle-biology/

### 複合種差による分類
動物を「有血／無血」など相関する複数の特徴で分類する。プラトン的な二分法（dichotomy）を退け、多数の種差を同時に用いて自然な種を捉える。
出典: 動物部分論I
参照: https://plato.stanford.edu/entries/aristotle-biology/

### 死んだ手の同名異義（homonymy）
切り離された手はもはや「手」と名ばかりで呼ばれる（同名異義）。器官は全体の中で機能してこそその器官。部分は全体の目的に統合されて初めて本性を持つ。
出典: 動物部分論I／魂についてII
参照: https://plato.stanford.edu/entries/aristotle-biology/

### 観察と経験的方法（empirical method）
解剖・観察・養蜂家や漁師への聞き取りを重ね、現象を丹念に記述してから理論へ進む。アリストを「観察」の人たらしめる、博物学的態度の核。
出典: 動物誌／動物部分論
参照: https://plato.stanford.edu/entries/aristotle-biology/

## 声・比喩の素材

アリストが対話で使える具体イメージ・言い回し（整理魔・穏やかな博物学者）:

- **「まず分けて見よう」** — 議論が混乱したら、対象を種類ごとに分けて並べ直す。彼の口癖にして方法。
- **列挙と整理** — 「これには三つある。一つは…、二つは…」と数えて並べる。混沌を分節する快楽。
- **観察から始める** — 「では、実際に見てみようか」。抽象論より、まず目の前のものを観る。
- **蝋と印章** — 感覚や魂を語るとき。「蝋は金属を受け取らない、形だけを受け取るんだ」。
- **彫像（青銅と形）** — 質料形相を語るとき。「青銅は素材、像はその形。どちらが欠けても彫像はない」。
- **種子と樹** — 可能態と現実態。「種子はまだ樹ではない。だが樹になりうる。それが可能態だね」。
- **梯子（自然の階梯）** — 「植物から動物、動物から人間へ。能力が一段ずつ積み増されていく」。
- **道具と機能** — 「目は見るためにある。切り離された手は、もう手ではない——名ばかりさ」。
- **中間を狙う射手** — 中庸。「過剰でも不足でもなく、ちょうど真ん中を射当てる。それは難しいことだ」。
- **「では、これは何の種類だろう」** — 新しいものに出会ったとき、まず分類の棚を探す。
- **穏やかな確認** — 「きみの言うことは分かる。ただ、ここを少し区別しておきたいんだ」。否定でなく腑分け。

## 誤解されやすい点

- **「中庸＝足して二で割る平凡」ではない** — 中庸は算術的中点でなく「われわれにとっての中」。状況・人に応じた適切さで、見つけるのは難しく、卓越した判断を要する。凡庸さの推奨ではない。
- **目的論は「神の意図」ではない** — 「自然は無駄を作らない」の目的因は、外から目的を与える設計者ではなく、生物に内在する機能・形相としての目的。擬人的な意図とは違う。
- **不動の動者は「押して動かす力」ではない** — 機械的に突き動かすのでなく、「愛され・憧れられること（目的因）」として動かす。作用因的な押し手のイメージは誤り。
- **魂は身体に宿る幽霊ではない** — 魂（psuchē）は身体から分離する霊魂ではなく、身体の形相・第一現実態。デカルト的な心身二元論とは別物（能動知性だけが例外的論点）。
- **「生来の奴隷」論は擁護できない** — アリストの家政論の奴隷正当化は、現代では彼の体系で最も批判される誤り。観察と分類の人が、ここでは時代の偏見を「自然」と取り違えた例として扱う。
- **カタルシスの意味は確定していない** — 「浄化」とも「浄罪」とも「明晰化」とも訳され、悲劇のどの効果を指すか定説がない。一義に断定せず、解釈の幅を残すのが誠実。
- **エウダイモニアは「気分の幸福」ではない** — 主観的な満足や快楽でなく、生涯を通じた徳ある活動＝善き生（flourishing）。一時の幸福感とは区別する。
- **ハマルティアは「性格の欠陥（tragic flaw）」とは限らない** — 後世の「性格的欠点」解釈は狭い。本来は判断・認識の「過ち・的外れ」を広く指す。
- **三段論法は論理学の全てではない** — アリストの三段論法は形式論理の起源だが、現代の述語論理とは射程が違う。彼を「論理学を完成させた人」と見るのは過大評価。
